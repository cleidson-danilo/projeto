# 📱 Aula Mobile - Ionic & Angular

Repositório dedicado ao armazenamento das atividades práticas e conceitos teóricos das aulas de **Desenvolvimento Mobile**, ministradas na **UNINASSAU**. O foco é o domínio do ecossistema Ionic e as novas funcionalidades do Angular Standalone.

## 🚀 Tecnologias e Ferramentas
* **Framework:** [Ionic Framework](https://ionicframework.com/)
* **Lógica:** [Angular](https://angular.io/) (Sintaxe @control flow)
* **Linguagem:** TypeScript
* **Ambiente:** Node.js v22+

## 📝 Conteúdo do "Caderno de Notas"
O projeto foi estruturado como um guia interativo (usando `ion-accordion`) para consulta rápida de:

1.  **Variáveis de Template (#):** Referência direta a elementos do DOM.
2.  **Lógica Condicional:** Uso de `@if`, `@else if` e `@else`.
3.  **Estruturas de Repetição:** Uso de `@for` com `track $index`.
4.  **Estado Vazio:** Implementação do `@empty` para feedback ao utilizador.
5.  **Eventos de Teclado:** Captura de dados com `keyup.enter`.
6.  **Binding:** Interpolação, Property e Event Binding.

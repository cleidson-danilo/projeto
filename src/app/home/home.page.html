 10. Eventos de Teclado e Manipulação de Listas

O que acontece aqui?

Quando você digita e aperta Enter, o Angular captura o evento através do (keyup.enter). Nós passamos a Variável de Template (#novoPais) para a função no TypeScript, que valida o texto e o adiciona a um Array. O @for detecta essa mudança na lista e atualiza a tela instantaneamente.

Código HTML (A Interface):

<!-- #input cria uma referência direta ao campo -->
<ion-input
  #input
  (keyup.enter)="add(input)"
  placeholder="Digite e dê Enter">
</ion-input>

Código TypeScript (A Lógica):

paises: string[] = []; // Nossa lista na memória

add(elemento: any) {
  const nome = elemento.value; // Pega o que foi digitado
  if (nome.length >= 3) {
    this.paises.push(nome);    // Adiciona ao Array
    elemento.value = "";       // Limpa o campo de texto
  }
}

Por que essa lógica é poderosa?

  • Variável de Template (#): Permite manipular o input sem precisar de ngModel (muito mais performático).
  • Filtro de Tecla: O .enter evita que o código rode a cada letra digitada, economizando processamento.
  • Validação Dinâmica: O @if dentro do @for permite criar alertas visuais para cada item da lista individualmente.

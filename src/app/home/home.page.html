  }
 7. Estruturas de Repetição (@for)

Significado: Loop ou laço de repetição.

Para que serve: Criar elementos HTML para cada item de uma lista automaticamente.

  @for (item of lista; track $index) {
    <p>{{ '{{' }} item {{ '}}' }}</p>
  }
 9. Mensagem Vazia (@empty)

Significado: Vazio.

Para que serve: Mostra um aviso se a lista estiver zerada [].

  @for (item of lista; track $index) { ... }
  @empty {
    <p>A lista está vazia.</p>
  }
